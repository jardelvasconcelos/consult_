SureFoam

Principio de Funcionamento

O sistema de dispensação Nordson® SureFoam mistura adesivo hot melt reativo e não reativo com gás inerte para criar uma mistura homogênea. Conforme o material é dispensado, o gás se expande para criar uma espuma de célula fechada. O material espumado pode ser aplicado em padrões de espiral ou de cordão usando pistolas distribuidoras Nordson. Os sistemas de espuma SureFoam são ideais para uma ampla variedade de aplicações adesivas, incluindo embalagem e montagem, como gabinetes e automotivo.

Dados Técnicos

  • Pressão máxima de gás: 1200 psi
  • Pressão hidráulica máxima: 1100 psi
  • Pressão máxima de ar: 90 psi
  • Temperatura de operação: 232°C

Capacidade

  • Fornece até 19kg/hr: podendo variar dependendo do nível de viscosidade do adesivo.
